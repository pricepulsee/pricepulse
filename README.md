# Price Pulse 💥

Агрегатор цен для белорусского рынка. Помогает покупателям понять — скидка реальная или нарисованная.

🌐 [pricepulse.by](https://pricepulse.by)

---

## Что умеет

- Сравнивает цены на товар у разных магазинов
- Показывает историю цен — видно, была ли «скидка» скидкой
- Даёт магазинам-партнёрам аналитику: где цена выше рынка, каких товаров не хватает в ассортименте

---

## Для магазинов

Подключение бесплатное. Размещение товаров, переходы и аналитика — бесплатно на весь период бета-тестирования.
